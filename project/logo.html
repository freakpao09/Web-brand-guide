Usage

Do

  • Use approved logo files from /assets/logos/ only.
  • Maintain clear space equal to the "S" height on every side.
  • Place on solid backgrounds — white, paper, ink, or forest-deep.
  • Use the icon mark only when space is constrained (≤ 64px).
  • Scale proportionally — never stretch width or height alone.

Don't

  • Recolor the mark — the "J" is always SEJ green.
  • Distort, stretch, rotate, or add effects like drop shadows.
  • Place over busy photography or low-contrast backgrounds.
  • Recreate or redraw the wordmark.
  • Set beside other logos without spacing equal to the "S" height.
